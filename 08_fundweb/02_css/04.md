<h1>Variáveis e Funções</h1>



As **variáveis CSS**, também chamadas de **Custom Properties**, permitem **armazenar valores reutilizáveis** que podem ser aplicados em múltiplas regras de estilo ao longo do código.

O principal benefício do uso de variáveis é a **centralização de valores**. Em vez de repetir o mesmo valor em diversas propriedades e seletores, ele é definido **uma única vez** e reutilizado sempre que necessário. Isso torna o código mais organizado, consistente e fácil de manter.

Em projetos reais, essa abordagem reduz significativamente o retrabalho. Por exemplo, ao alterar a **cor primária** de um site, basta modificar o valor da variável correspondente, e a mudança será refletida automaticamente em todos os pontos onde ela for utilizada.

Além disso, variáveis CSS tornam o código mais **legível e semântico**, já que valores genéricos passam a ter nomes descritivos, facilitando o entendimento do propósito de cada estilo e diminuindo a chance de erros durante manutenções.

<br />

<h3>1.1. Definição de uma variável CSS



A sintaxe básica para definir uma variável CSS é:

```css
:root {
  --bgcolor: #73a4f4;
}
```

As variáveis são normalmente declaradas no seletor `:root`, que representa o elemento raiz do documento (a tag `<html>`). Dessa forma, a variável se torna **global**, podendo ser utilizada em qualquer parte da página.

<br />

<h3>1.2. Regras importantes sobre variáveis CSS</h3>



- Toda variável **deve começar com dois hífens (`--`)**
- O nome da variável é **case-sensitive** (diferencia maiúsculas de minúsculas)
- O valor pode ser qualquer valor CSS válido (cores, tamanhos, fontes, espaçamentos, etc.)
- Uma variável CSS pode utilizar o valor de **outra variável**

<br />

<h3>1.3. Vantagens do uso de variáveis CSS</h3>



- Centralização e padronização de valores
- Facilidade de manutenção e refatoração
- Código mais limpo e semântico
- Redução de erros em projetos grandes
- Ideal para temas, cores, espaçamentos e design systems

O uso de variáveis CSS é uma prática recomendada em projetos modernos e constitui a base para a criação de **temas**, **modos claro/escuro** e **sistemas de design reutilizáveis**.

<br />

<h2>2. Funções</h2>



As **funções CSS** permitem **calcular valores**, **manipular cores**, **aplicar transformações** e **criar efeitos dinâmicos**, tornando o código mais flexível e expressivo. Em vez de trabalhar apenas com valores fixos, elas possibilitam comportamentos adaptáveis, essenciais para layouts responsivos e interfaces modernas.

De forma geral, uma função CSS segue este padrão:

```css
propriedade: nome-da-funcao(valor);
```

As funções são amplamente utilizadas para:

- Ajustar tamanhos de forma dinâmica
- Trabalhar com cores de maneira mais precisa
- Criar animações e transformações
- Aplicar efeitos visuais
- Melhorar a responsividade sem cálculos manuais

Na prática, funcionam como **ferramentas internas do CSS**, resolvendo problemas comuns do dia a dia de forma mais elegante e reutilizável.

<br />

<h3>2.1. Função para Variáveis</h3>



Para utilizar uma variável em uma propriedade CSS, utiliza-se a função `var()`:

```css
var(--nome_da_variavel)
```

A função `var()` recebe o nome da variável como parâmetro e retorna o valor associado a ela, permitindo reaproveitamento e centralização de configurações no código.

<br />

<img src="https://i.imgur.com/DSBasbH.png" title="source: imgur.com" width="4%"/>**CSS - Exemplo:**

```css
:root {
	--bgcolor: #73a4f4;
}
section{
	background-color: var(--bgcolor);
}

```

Neste exemplo, a variável `--bgcolor` é definida no seletor `:root`, tornando-se global, e aplicada à propriedade `background-color` de todos os elementos `<section>`. Qualquer alteração no valor da variável será refletida automaticamente em todos os locais onde ela for utilizada.

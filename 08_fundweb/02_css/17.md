# Transições (Transitions)



As **transitions** em CSS são uma das formas mais simples e poderosas de criar animações suaves em uma página web. Elas permitem que alterações nos estilos de um elemento ocorram de maneira gradual, em vez de imediata, proporcionando uma experiência mais fluida e agradável ao usuário.

<br />

## **1. O Que São CSS Transitions?**



Por padrão, quando um estilo CSS muda — como a cor de um botão, o tamanho de um texto ou a posição de um elemento — a mudança acontece instantaneamente. Com as **transitions**, você define uma duração e um comportamento para essa mudança, permitindo que o navegador faça a transição de um estado para outro de forma suave.

Em outras palavras: as transitions animam o **processo de mudança de uma propriedade CSS**.

Por exemplo, imagine um botão que muda de cor quando o cursor passa sobre ele. Sem transitions, a mudança é imediata; com transitions, a cor vai mudando suavemente, como um pequeno fade.

<br />

## **2. Características Principais**



As transitions têm algumas características importantes:

- **São fáceis de implementar:** bastam poucas linhas de CSS.
- **Dependem de uma mudança de estado:** geralmente ativadas por pseudoclasses como `:hover`, `:focus` ou `:active`.
- **Afetam apenas propriedades animáveis:** nem todas as propriedades CSS podem ser animadas (por exemplo, `display` não é).
- **Podem ter duração, atraso e curva de aceleração personalizadas.**

<br />

## **3. Propriedades Fundamentais da Transition**



Existem **quatro propriedades principais** que controlam o comportamento das transitions:

### ✅ `transition-property`**



Define **qual propriedade CSS será animada**.

**Exemplos:**

```css
transition: background-color;
```

Realiza a transição apenas na propriedade **`background-color`**.

```css
transition: all;
```

Anima todas as propriedades que mudarem.

<br />

### ✅ **`transition-duration`**



Define **quanto tempo** a transição deve durar. O valor pode ser em **segundos (`s`)** ou **milissegundos (`ms`)**.

**Exemplo:**

```css
transition: all 0.5s;
```

<br />

### ✅ ** `transition-timing-function`**



Controla **a velocidade da animação ao longo do tempo**, ou seja, se ela começa lenta, acelera e desacelera no final, e assim por diante.

**Valores comuns:**

- `ease` → começa devagar, acelera no meio e desacelera no fim (padrão)
- `linear` → velocidade constante
- `ease-in` → começa lenta e acelera
- `ease-out` → começa rápida e desacelera
- `ease-in-out` → lenta no início e no fim

**Exemplo:**

```css
transition: all 0.5s ease-in-out;
```

<br />

### ✅ ** `transition-delay`**



Define **quanto tempo esperar** antes de iniciar a transição.

**Exemplo:**

```css
transition: all 0.5s ease-in-out 0.2s;
```

<br />

Vamos aplicar uma transição em um botão que muda de cor e tamanho quando o cursor do mouse estiver sobre o botão:

<img src="https://i.imgur.com/O5NjoiA.png" title="source: imgur.com" width="5%"/>**Exemplo:**

```html
<button class="botao">Clique Aqui</button>
```

<img src="https://i.imgur.com/DSBasbH.png" title="source: imgur.com" width="4%"/>**CSS:**

```css
.botao {
  background-color: #6200ea;
  color: white;
  padding: 12px 24px;
  border: none;
  border-radius: 8px;
  cursor: pointer;
  transition: background-color 0.3s ease, transform 0.3s ease;
}

.botao:hover {
  background-color: #3700b3;
}
```

<img src="https://i.imgur.com/3HPZ0By.png" title="source: imgur.com" width="3%"/>**Resultado:**

![](https://i.imgur.com/IS49iN6.gif)

**Como funciona:**

- Quando o mouse passa sobre o botão (`:hover`), a cor de fundo e o tamanho mudam.
- A mudança acontece de forma suave, durando 0.3 segundos.
- O `ease` faz a animação começar e terminar devagar, o que deixa o movimento natural.

<br />

## **4. Dica: Usando `transition` com Vários Elementos**



Você pode aplicar transitions em praticamente qualquer elemento: imagens, textos, containers, cards, entre outros. Por exemplo, para criar um efeito de destaque em uma imagem ao passar o mouse:

<img src="https://i.imgur.com/O5NjoiA.png" title="source: imgur.com" width="5%"/>**Exemplo:**

```html
<img
       src="https://imgur.com/QkPd4Nl.png"
       alt="Imagem do Blog"
>
```

<img src="https://i.imgur.com/DSBasbH.png" title="source: imgur.com" width="4%"/>**CSS:**

```css
img {
  filter: brightness(0.8);
  transition: filter 0.4s ease;
}

img:hover {
  filter: brightness(1);
}
```

<img src="https://i.imgur.com/3HPZ0By.png" title="source: imgur.com" width="3%"/>**Resultado:**

![](https://imgur.com/ijtYWSp.gif)

Assim, a imagem fica mais clara suavemente quando o usuário passa o cursor.

<br />

## **5. Quando Usar Transitions**



Use transitions para:

- Criar **feedback visual** em botões, menus e links.
- Dar **destaque a interações** sutis, como o foco em um campo de formulário.
- Tornar o **layout mais dinâmico e fluido**, sem exageros.

Mas evite:

- Animar muitas propriedades simultaneamente (pode prejudicar o desempenho).
- Usar durações muito longas (acima de 1s) em elementos pequenos.

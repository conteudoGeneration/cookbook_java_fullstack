# Animações (Animations)



## **1. O Que São Animações CSS**



Uma **animação CSS** é um conjunto de regras que define **como e quando** um elemento deve mudar ao longo do tempo. Diferente das *transitions*, que dependem de um evento (como um `hover`), as animações podem ser **automáticas e contínuas**, iniciando assim que a página é carregada ou em momentos específicos do ciclo de interação do usuário.

Essas animações são definidas por meio da regra especial `@keyframes`, que descreve as etapas da animação, e por propriedades aplicadas ao elemento que determinam a duração, repetição e forma de execução.

<br />

## **2. O que são as animações com `@keyframes`?**



No capítulo anterior, aprendemos que as **transições (`transition`)** são ótimas para criar movimentos simples, que acontecem **quando algo muda** — por exemplo, ao passar o mouse sobre um botão.

Mas e se quisermos criar **movimentos automáticos**, que **acontecem sozinhos**, ou que tenham **várias etapas**?

É aí que entra o **`@keyframes`**, uma ferramenta do CSS que permite **definir passo a passo** o comportamento de uma animação.

A palavra ***keyframe*** significa “quadro-chave”. Cada quadro representa um **momento específico da animação** — como se fosse uma cena em um filme.

**Exemplo**

![](https://imgur.com/o15tsVD.png)

- No **início (0% do tempo)**, a imagem está na posição original (zero).
- No **meio (50% do tempo)**, a imagem é deslocada **1.25rem** (20 px) para cima.
- No **fim (100% do tempo)**, a imagem volta para sua posição original.

Esses pontos de controle (0%, 50%, 100%) são os “quadros-chave”.

<br />

## **3. Estrutura básica de uma animação**



Para criar uma animação com `@keyframes`, usamos **duas etapas principais**:

1. Criar a sequência de movimento usando `@keyframes`;
2. Aplicar essa animação a um elemento usando propriedades CSS.

<br />

<img src="https://i.imgur.com/O5NjoiA.png" title="source: imgur.com" width="5%"/>**Exemplo:**

```html
<div class="container">
  <button class="botao">Clique Aqui</button>
</div>
```

<img src="https://i.imgur.com/DSBasbH.png" title="source: imgur.com" width="4%"/>**CSS:**

```css
@keyframes mudarCor {
  0% {
    background-color: #6b36c8;
  }
  100% {
    background-color: #9a7bd1;
  }
}

.container {
  width: 100px;
  height: 100px;
  background-color: #6b36c8;
  animation: mudarCor 5s;
}
```

<img src="https://i.imgur.com/3HPZ0By.png" title="source: imgur.com" width="3%"/>**Resultado:**

![](https://i.imgur.com/yVgOaJZ.gif)

**Como funciona este exemplo:**

1. A regra `@keyframes` define a animação chamada **mudarCor**.
    - No **início (0%)**, a cor inicial é roxa escura (`#6b36c8`).
    - No **fim (100%)**, ela muda para a cor roxa clara (`#9a7bd1`).
2. O elemento com a classe `.container` recebe essa animação através da propriedade:
    - `animation`: que indica o nome da animação (“mudarCor”) e o tempo de duração da aimação ( 5 segundos).

O resultado é uma caixa que muda de cor automaticamente ao carregar a página.

<br />

## **4. Adicionando mais etapas à animação**



Nem sempre a animação precisa ter apenas um começo e um fim. Podemos criar **várias etapas intermediárias**, indicando porcentagens diferentes.

Veja este exemplo com **três cores**:

<img src="https://i.imgur.com/O5NjoiA.png" title="source: imgur.com" width="5%"/>**Exemplo:**

```html
<div class="container">
  <button class="botao">Clique Aqui</button>
</div>
```

<img src="https://i.imgur.com/DSBasbH.png" title="source: imgur.com" width="4%"/>**CSS:**

```css
@keyframes cicloDeCores {
  0% {
    background-color: #6b36c8; /* roxo escuro */
  }
  50% {
    background-color: #9a7bd1; /* roxo médio */
  }
  100% {
    background-color: #e9dcfa; /* roxo claro */
  }
}

.container {
  width: 100px;
  height: 100px;
  background-color: #6b36c8;
  animation: cicloDeCores 5s;
}
```

<img src="https://i.imgur.com/3HPZ0By.png" title="source: imgur.com" width="3%"/>**Resultado:**

![](https://imgur.com/1860AoQ.gif)

Agora a cor **muda suavemente entre três tons de roxo**, criando um efeito de transição contínua.

<br />

## **5. Tornando a animação contínua**



Por padrão, a animação acontece **apenas uma vez**. Mas podemos fazer com que ela **repita infinitamente** usando a opção `infinite`.

```css
.container {
  animation: cicloDeCores 3s infinite;
}
```

Agora, a animação recomeça assim que termina — criando um movimento cíclico.

<br />

## **6. Controlando a direção da animação**



Podemos também fazer a animação **alternar de direção** com a propriedade `animation-direction`.

```css
.container {
  animation: cicloDeCores 3s infinite alternate;
}
```

Nesse caso, a cor vai do tom escuro ao claro e depois volta do claro ao escuro, repetindo continuamente.
